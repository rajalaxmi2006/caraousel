# Simple Carousel

A basic and responsive carousel built using HTML, CSS, and JavaScript.  
It displays 4–5 slides with smooth transitions and simple navigation.

## Features
- 3-4 slides
- Smooth sliding animation
- Previous and Next buttons
- Responsive design
- Lightweight and easy to customize

## Built With
- HTML
- CSS
- JavaScript

## Project Structure
simple-carousel/
│
├── index.html
├── style.css
└── script.js

## Getting Started

1. Clone the repository
git clone https://github.com/your-username/simple-carousel.git

2. Open `index.html` in your browser  
No installation or dependencies required.

## Usage
- Edit slide content in `index.html`
- Customize styles in `style.css`
- Modify carousel behavior in `script.js`
